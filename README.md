#How to run code
1. Click green code button and dowload code as zip
2. Unzip the folder to your desktop
3. Dowload an IDE that can run java, Ex. Intellij idea Community edition https://www.jetbrains.com/idea/download/#section=windows
4. Dowload the java plug in Intellij Idea
5. Open the unziped folder in Intellij
6. If arrow in top right is grey instead of green click on the hammer
7. Otherwise run the code and type inputs in the console when promted
